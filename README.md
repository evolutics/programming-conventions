# Programming Conventions

## Naming

### Terms that go together

- absent, present
- acquire, release
- actual, desired
- add, remove
- after, before
- agent, controller
- allow, deny
- close, open
- create, delete
- demote, promote
- destination, source
- failure, success
- infix, prefix, suffix
- load, save
- new, old
- next, previous
- parse, print
- pop, push
- pull, push
- read, write
- set up, tear down
- start, stop

## Programming languages

### Bash

Follow this [style guide](https://google.github.io/styleguide/shellguide.html).

### Python

Follow this [style guide](https://peps.python.org/pep-0008/).

### Rust

Follow this
[naming guide](https://rust-lang.github.io/api-guidelines/naming.html).

It is fine to panic with `unwrap()` to signal bugs.
